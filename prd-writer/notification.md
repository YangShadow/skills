### 3. 触发条件

| 条件类型 | 具体规则 |
|---------|---------|
| 触发时机 | [用户完成某操作 / 某时间点 / 某状态变化] |
| 触发用户范围 | [全量 / 按标签圈选 / 特定行为用户] |
| 触发频次限制 | [单用户最多触发 N 次 / 每日上限 / 仅触发一次] |
| 冷却期 | [触发后 N 天内不再触发] |
| 前置条件 | [需开启通知权限 / 需登录 / 需满足 X 条件] |

---

### 4. 展示逻辑

| 规则 | 说明 |
|------|------|
| 优先级 | 与其他通知并发时，本通知的优先级排序 |
| 是否打断用户 | 强打断（弹窗需手动关闭）/ 弱提示（Toast 自动消失） |
| 自动消失 | 是（N 秒后）/ 否 |
| 关闭后是否再次出现 | 是（下次满足条件时）/ 否（永久关闭）/ 最多再出现 N 次 |
| 未读角标 | 是 / 否 |
