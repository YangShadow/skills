# stock-research Skill

## 触发方式

```
/stock-research macro   <主题>    # 仅宏观分析（Fed/CPI/PPI/地缘等）
/stock-research fund    <ticker>  # 仅基本面+财报解读
/stock-research tech    <ticker>  # 仅技术分析（趋势+支撑阻力）
/stock-research full    <ticker>  # 三层完整分析
```

## Step 1：意图识别与路由

解析用户输入的模式参数（macro / fund / tech / full），确定激活哪些分析层。
若未提供模式参数，询问用户选择。

## Step 2：数据采集（调用 web-access skill）

### 数据源分级政策（严格执行）

**S级 — 一手官方数据（直接采用，最高置信度）**
- 货币政策：federalreserve.gov（利率决议/FOMC纪要/点阵图）
- 行政政策：whitehouse.gov / ustr.gov（关税/贸易）/ treasury.gov（财政/债券）
- 经济数据：bls.gov（CPI/PPI/非农）/ bea.gov（GDP/PCE）/ eia.gov（原油库存）
- 市场数据：sec.gov/EDGAR（10-K/10-Q/8-K）/ cmegroup.com（FedWatch利率概率）

**A级 — 权威财经媒体（注明来源后采用）**
bloomberg.com / reuters.com / ft.com / wsj.com / investing.com / tradingview.com / edition.cnn.com

**B级 — 情绪参考（仅辅助，不进入核心结论）**
- polymarket.com（事件概率）
- x.com / twitter.com（需≥3个独立KOL印证，且观点方向一致）

**丢弃规则：**
- 匿名来源、单一社媒说法、无S/A级支撑的观点 → 标注「待验证」或直接丢弃
- B级信息若与S/A级结论矛盾 → 以S/A级为准，B级仅在「市场情绪」节呈现

### 宏观层采集（macro / full 模式）

按以下优先级顺序抓取：
1. federalreserve.gov → 最新利率决议、FOMC纪要摘要、点阵图中位数
2. bls.gov → 最新CPI/PPI数据，同比/环比变化
3. bea.gov → PCE、GDP最新值
4. eia.gov → 原油库存、天然气数据（地缘相关时）
5. cmegroup.com/fedwatch → 市场隐含降息概率
6. whitehouse.gov / ustr.gov → 相关政策声明（贸易/关税主题时）
7. bloomberg.com / reuters.com → 事件背景补充（A级）
8. polymarket.com → 事件结果概率（B级，仅情绪参考）
9. x.com → 搜索相关话题下≥3个金融KOL观点（B级）

### 基本面层采集（fund / full 模式）

1. sec.gov/EDGAR → 搜索 ticker，抓取最新 10-K（年报）和 10-Q（季报）
2. 提取：营收、净利润、经营现金流、自由现金流（近5年）
3. 提取：费用明细（销售/管理/研发/财务）
4. 提取：存货数据、资产负债表关键项
5. bloomberg.com / investing.com → 分析师评级和目标价（A级，辅助参考）

### 技术层采集（tech / full 模式）

1. tradingview.com → 日线/周线图表，抓取以下指标当前值：
   - RSI（14日）：数值 + 是否存在背离形态
   - OBV：趋势方向（上升/下降/走平）
   - EMA（21/50日）和 SMA（50/200日）：价格与均线位置关系
   - 布林带：宽度状态（挤压/扩张）+ 价格位置
   - VIX（如为美股分析）：当前值
2. investing.com → 成交量数据（20日均量对比）
3. 手动计算：斐波那契回撤位（需识别最近主要波段高低点）

## Step 3：分析推演

### 宏观层分析
- 构建逻辑因果链：数据事实 → 传导机制 → 市场影响
- 例：CPI超预期 → Fed维持鹰派 → 利率预期上升 → 成长股估值承压
- 地缘主题：事件 → 商品价格 → 通胀传导 → 政策响应

### 基本面层分析（7步框架）

**第1步：报告识别**
确认报告类型（季报/半年报/年报）、报告期间、公司基本信息。

**第2步：营收净利润5年趋势**
- 计算各年同比增长率
- 计算CAGR（复合年增长率）= (最新值/5年前值)^(1/5) - 1
- 判断增长稳定性：是否出现连续负增长或大幅波动

**第3步：现金流深度分析**
- 经营性现金流/净利润比：>80%为健康，<50%需警惕
- 自由现金流 = 经营性现金流 - 资本支出
- 连续3年以上自由现金流为正 = 强财务质量信号

**第4步：费用结构分析**
- 各费用占营收比的5年趋势
- 销售费用增速 vs 营收增速对比（效率判断）
- 研发费用持续投入 = 护城河信号

**第5步：存货风险评估**
- 存货周转率 = 营业成本 / 平均存货（次/年）
- 同比下降>30% → 🔴红色预警
- 存货占总资产比 > 行业均值+50% → 🔴红色预警

**第6步：异常指标识别**
对所有同比波动>30%的指标逐项分析原因，评估可持续性。

**第7步：综合评估**
- 财务健康度评级：优秀 / 良好 / 一般 / 较差
- 投资建议：买入 / 持有 / 观望 / 回避 + 理由

**8类暴雷资产扫描：**
逐项检查并标注风险程度（高/中/低/无）：
1. 货币资金：账面资金多但大量借贷 → 警惕
2. 应收账款：与营收同步激增 → 警惕
3. 存货减值：高存货行业 → 评估跌价准备
4. 固定资产：高专用性资产 + 客户高度集中 → 警惕
5. 无形资产：占比过高且难以核实 → 警惕
6. 商誉：收购溢价过高 → 评估减值风险
7. 长期股权投资：被投资方经营状况
8. 净现金：现金+短期投资-有息负债，为负则警惕

**风险预警评级：**

🔴 红色预警触发条件（任一触发即标注）：
- 营收同比 < 0%
- 净利润率同比下降 > 30%
- 经营性现金流为负
- 净利润与经营现金流差异 > 50%
- 资产负债率 > 70%
- 流动比率 < 1.0

🟡 黄色预警触发条件：
- 营收增速低于行业均值
- 毛利率持续下降
- 费用增速 > 营收增速
- 应收账款占比过高

### 技术层分析

**趋势判断：**
- RSI < 30 → 超卖，观察看涨背离
- RSI > 70 → 超买，观察看跌背离
- 看涨背离：价格创新低但RSI未创新低 → 底部反转先导信号
- 看跌背离：价格创新高但RSI未创新高 → 顶部反转先导信号
- 隐藏看涨背离：价格高点抬升但RSI低点下移 → 趋势延续信号
- OBV与价格同向 → 趋势健康；OBV背离价格 → 趋势衰竭预警
- 价格站上21EMA → 短期偏多；跌破21EMA → 短期偏空
- 价格站上200SMA → 长期牛市结构
- 金叉（50SMA上穿200SMA）→ 长期看涨信号
- 死叉（50SMA下穿200SMA）→ 长期看跌信号
- VIX > 50 → 极端恐慌，历史上接近市场底部
- ZBT从<0.40快速拉升突破0.615 → 强烈看涨信号

**支撑阻力判断：**
- 识别最近3-6个月的主要波段高点（阻力）和低点（支撑）
- 趋势线：连接≥3个触点才视为有效
- 布林带挤压（带宽收窄至极值）→ 预警即将突破行情
- 突破确认：收盘价突破S/R区域 + 成交量≥20日均量×150% → 有效突破
- 假突破：突破时成交量低迷 → 大概率回踩
- 斐波那契关键回撤位：38.2% / 50% / 61.8%

**共振评级（分数制）：**

统计以下信号中多头信号数量，计算得分 X/10：

| 信号 | 多头条件 |
|------|---------|
| RSI | 看涨背离或从超卖区回升 |
| OBV | 与价格同向上升或看涨背离 |
| 均线 | 价格站上21EMA且50SMA向上 |
| 成交量 | 突破时放量≥均量150% |
| 布林带 | 挤压后向上突破 |
| VIX | 从高位回落（>40→回落中） |
| ZBT | 突破0.615 |
| 趋势线 | 价格守住上升趋势支撑线 |
| 斐波那契 | 价格在61.8%回撤位获支撑 |
| K线形态 | 出现锤子/吞没/早晨之星等反转形态 |

评级解读：
- 8-10分：高概率多头信号，强烈关注做多机会
- 5-7分：中等概率，需等待更多确认
- 0-4分：空头占优或信号混乱，保持观望
