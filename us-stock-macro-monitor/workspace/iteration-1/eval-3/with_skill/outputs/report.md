## 地缘政治风险

### 1. 中东危机（最高优先级）

**美伊冲突升级：**
- 美国于3月下旬向伊朗发出48小时最后通牒，要求重新开放霍尔木兹海峡（全球约20%石油运输通道）
- 伊朗袭击卡塔尔、沙特、科威特的LNG和炼油设施
- 卡塔尔Ras Laffan综合体遭重创，全球LNG产量下降
- 布伦特原油飙升至**约112美元/桶**

**区域局势恶化：**
- 导弹袭击以色列核设施附近区域
- 美国及盟友在阿联酋、科威特基地遭袭
- 沙特驱逐伊朗外交官
- 主要航运公司绕行非洲好望角，运输时间和成本大幅增加

### 2. 俄乌冲突

- 战事陷入僵局，双方均未取得决定性胜利
- 黑海基础设施持续面临港口、炼油厂和运输网络袭击风险
- 欧盟准备在美国支持不确定的情况下独立支持乌克兰
- RANE预测：双方2026年初抗拒让步，但压力增大可能促成下半年谈判

### 3. 台海紧张局势

- 被列为2026年**一级闪点**（最高溢出风险）
- 中国对台施压持续，威胁全球供应链崩溃（特别是半导体和电子元件）
- 中国加强主权声索和区域军事化
- 外交关系委员会评估：台海具有"高升级潜力的大国纠缠风险"

### 4. 其他地缘政治风险

| 风险因素 | 状态 |
|---------|------|
| 中美竞争 | 加剧；贸易脱钩和科技限制持续 |
| 美国关税政策 | 面临法律挑战；与主要贸易伙伴争端持续 |
| 全球供应链 | 多重同时中断下承受严重压力 |
| 国防支出 | 全球大幅增加，欧洲部分国家目标2035年达GDP的3.5% |

## 市场情绪与预测

### 专家评估

**RANE（2026年1月）：** "全球波动性将保持高位，因加剧的大国竞争和未解决的地区冲突相互碰撞。"

**外交关系委员会调查：** 将俄乌/NATO事件风险评为"均等机会"，台海评为2026年最高升级潜力。

### 经济与市场影响

- **石油/LNG价格：** 因中东中断而大幅上涨
- **货运费率：** 因航线改道和战争风险保险费而急剧上升
- **通胀压力：** 能源成本飙升威胁推动通胀超过目标
- **航空货运瓶颈：** 航空公司绕开冲突区飞行
- **投资组合多样化：** 投资者 increasingly 寻求避险资产（瑞士法郎、黄金、大宗商品）

## 总结

当前美股宏观环境面临**通胀回升**与**地缘政治风险**的双重压力：

1. **通胀方面：** 2月CPI虽维持2.4%，但3月预期大幅跳升，Fed已上调全年通胀预测至2.7%，显示对通胀粘性的担忧。

2. **地缘政治：** 中东美伊冲突是最紧迫风险，霍尔木兹海峡危机可能引发全球能源危机；台海和俄乌局势持续构成结构性风险。

3. **政策应对：** Fed维持利率不变，采取"观望"态度，但内部分歧显现（1票反对），市场定价显示对降息预期大幅降温。

4. **市场影响：** VIX维持高位（26+），美债收益率上升，美元反弹，反映风险厌恶情绪升温。

**投资者应密切关注：** 4月10日CPI数据、中东局势演变、Fed官员讲话。
